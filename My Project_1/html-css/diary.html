DAY1

花了點時間複習，當個兵甚麼東西都忘了。
先做出了首頁，因為沒有甚麼美感所以做了較簡單的風格。
上面做了四個連結(包含標題大字)，都可以點擊跳轉到頁面，
但每個頁面都還沒編輯。

連結上使用 text-decoration: none 取消原有的底線，
再透過 a:hover 屬性編輯滑鼠經過連結的的表現，
最後 a:active 屬性編輯滑鼠點擊連後的樣子。
